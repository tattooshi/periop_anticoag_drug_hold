周術期における抗凝固薬・抗血栓薬の休薬期間を商品名・一般名から即座に検索できるオフラインFlutterアプリ
# Perioperative Anticoagulation Management App

A lightweight Flutter-based offline application to quickly search perioperative anticoagulation and antithrombotic drug withdrawal periods by product name or generic name.

## ✨ Features

- Offline usage (no network required)
- Search by:
  - Product name (priority)
  - Generic name (fallback)
- Automatic Katakana normalization for Japanese input
- Lightweight APK (~20MB)
- Optimized for clinical workflow
